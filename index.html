Operations
Key Concepts
Determinant (2×2)
det(A) = ad − bc for A = [[a,b],[c,d]]
Determinant (3×3) — Cofactor Expansion along Row 1
det(A) = a₁₁·C₁₁ + a₁₂·C₁₂ + a₁₃·C₁₃
where C₁₁ = +(a₂₂a₃₃ − a₂₃a₃₂), C₁₂ = −(a₂₁a₃₃ − a₂₃a₃₁), C₁₃ = +(a₂₁a₃₂ − a₂₂a₃₁)
Minor Mij
The determinant of the submatrix formed by deleting the i-th row and j-th column.
Cofactor Cij
Cij = (−1)i+j · Mij
Sign pattern: + − +
                 − + −
                 + − +
Adjugate (Adjoint) adj(A)
Transpose of the cofactor matrix: adj(A) = [Cij]ᵀ
Inverse A⁻¹
A⁻¹ = adj(A) / det(A)  (exists only when det(A) ≠ 0)
Trace
tr(A) = sum of diagonal elements = a₁₁ + a₂₂ [+ a₃₃]
Transpose AT
Rows become columns: (Aᵀ)ij = Aji
Rank
Maximum number of linearly independent rows/columns. For 2×2: rank = 2 if det≠0, else rank = 1 (or 0 if all zeros). For 3×3: rank = 3 if det≠0, else determined by largest non-zero minor.
Singular vs Invertible
A matrix is singular if det(A) = 0 (no inverse exists).
A matrix is invertible/non-singular if det(A) ≠ 0.
Matrix Multiplication (A×B)
(AB)ij = Σ Aik · Bkj — row i of A dotted with column j of B.
Properties
det(AB) = det(A) · det(B)
det(Aᵀ) = det(A)
det(kA) = kⁿ · det(A) for n×n matrix
(AB)⁻¹ = B⁻¹ · A⁻¹
(Aᵀ)⁻¹ = (A⁻¹)ᵀ
