# Matrix Solver

A modern interactive **Matrix Calculator & Linear Algebra learning
tool** built with pure **HTML, CSS, and JavaScript**.

This project allows users to compute determinants, inverses, minors,
cofactors, and perform matrix operations while also showing
**step‑by‑step solutions** to help understand the math behind the
results.

------------------------------------------------------------------------

## Features

### Matrix Calculations

-   Determinant calculation
-   Trace calculation
-   Matrix transpose
-   Adjugate matrix
-   Matrix inverse
-   Scalar multiplication

### Linear Algebra Tools

-   Minors and cofactors (3×3 matrices)
-   Step-by-step determinant solution
-   Automatic singular / invertible detection
-   Identity matrix generator
-   Random matrix generator

### Matrix Operations

-   Matrix addition (A + B)
-   Matrix subtraction (A − B)
-   Matrix multiplication (A × B)
-   Reverse multiplication (B × A)

### Learning & Theory

Built‑in explanations for:

-   Determinants
-   Minors & cofactors
-   Matrix inverse
-   Trace
-   Rank
-   Matrix multiplication
-   Important matrix properties

------------------------------------------------------------------------
